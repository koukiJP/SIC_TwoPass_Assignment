# Simple SIC Two Pass Assignment By C++

System Program Assignment 1

```
.
├── README.md
├── SourceSIC                    // Dir for saving SIC Source Program
│   ├── COPY.txt
│   ├── COPY_Ver_TextBook.txt
│   ├── COUNT.txt
│   └── SUM.txt
├── TargetCode                  // Dir for saving Target Code
│   ├── COPY
│   ├── COUNT
│   └── SUM
├── asm_listing                 // Dir for saving Asm_listing
│   ├── COPY
│   ├── COUNT
│   └── SUM
├── interTables                 // Dir for saving intermediate files
│   ├── OPTABLE.txt
│   ├── intermediate.txt
│   └── symtab.txt
├── main.cpp                    // main cpp file
└── runc.sh                     // Simple bash file to execute C/C++
```

---

## Usage

```sh
git clone https://github.com/koukiJP/SIC_TwoPass_Assignment.git
cd SIC_TwoPass_Assignment
./runc.sh main.cpp
# chose the Source Program you want to translate into Target Code :
```
